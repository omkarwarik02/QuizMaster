# 🧠 QuizMaster

A fun and interactive quiz app built with React and Tailwind CSS.

## 🚀 Live Demo
[Click here to play QuizMaster](https://quiz-master-tau-lovat.vercel.app/)

## 📸 Features

- 🎯 Multiple categories — Geography, Math, Science
- ⏱️ 10 second timer per question
- 📊 Score tracking
- 🎨 Beautiful UI with Tailwind CSS
- 🔄 Play Again option
- 📱 Fully responsive

## 🛠️ Built With

- [React](https://react.dev/) — UI library
- [Vite](https://vitejs.dev/) — Build tool
- [Tailwind CSS](https://tailwindcss.com/) — Styling

## 💡 Concepts Used

- Components & JSX
- Props
- useState
- useEffect
- useContext
- Conditional rendering
- Array methods (.map, .filter)
- Event handling

## ⚙️ Run Locally

Clone the project:
```bash
git clone https://github.com/yourusername/quizmaster.git
```

Go to project directory:
```bash
cd quizmaster
```

Install dependencies:
```bash
npm install
```

Start the server:
```bash
npm run dev
```
